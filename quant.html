🔌WorkBuddy MCP 接入

把 HKUDS Vibe-Trading 作为 MCP 服务器接入 WorkBuddy 专家，让 LLM 自动调用量化能力。① 安装 Vibe-Trading MCP 包，② 在 ~/.workbuddy/mcp.json 加 vibetrading 条目，③ 配置 OPENAI_API_KEY + LONGBRIDGE_TOKEN 即可让本专家调用回测 / 因子 / α 动物园。

// C:/Users/29096/.workbuddy/mcp.json { "mcpServers": { "vibetrading": { "command": "python", "args": ["-m", "vibetrading.mcp"], "env": { "OPENAI_API_KEY": "sk-...", "LONGBRIDGE_TOKEN": "..." } } } }

▶ 接入后，本专家即可在对话中触发：vibe research "招商银行 30 天回测"、vibe bench --strategies factor_etf、vibe shadow --capital 100000 等命令。
